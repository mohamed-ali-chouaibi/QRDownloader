# QR Studio

Create beautiful, printable QR codes for anything.

## Features

- **Wi-Fi QR Codes**: Generate QR codes that automatically connect devices to your Wi-Fi network
- **Website QR Codes**: Create QR codes that link to any website or URL
- **Professional PDFs**: Download high-quality, printable PDF files
- **Privacy First**: All data is processed directly in your browser - nothing is stored on servers
- **Modern Design**: Beautiful gradient interface with smooth animations

## How to Use

### Creating a Wi-Fi QR Code

1. Select "Wi-Fi Network" from the card options
2. Enter your network details:
   - Wi-Fi Network Name (SSID)
   - Password
   - Security Type (WPA/WPA2, WEP, or No Password)
3. Click "Generate & Preview"
4. Download the PDF to print and share

### Creating a Website QR Code

1. Select "Website URL" from the card options
2. Enter your website details:
   - Website Address (URL) - must include http:// or https://
   - Custom Title (optional) - shown on the PDF
3. Click "Generate & Preview"
4. Download the PDF to print and share

## PDF Features

The generated PDFs include:
- Clean white background
- Large, scannable QR code with gradient border
- Network name or website title prominently displayed
- Password display (for Wi-Fi) with accent styling
- Clear scanning instructions
- Professional footer branding

Perfect for:

- Retail displays

## Technical Details

- **QR Code Library**: qrcode.js
- **PDF Generation**: jsPDF
- **Error Correction**: Level L (7% recovery)
- **QR Code Size**: 256x256 pixels (display), 95mm (PDF)
- **PDF Format**: A4 portrait
